# From Scratch

**From Scratch** is a frontend-only MVP app that connects fine artists and textile craftsmen with surplus, offcut, and rescued materials — reducing waste one exchange at a time.

All connections between material givers and receivers are facilitated by an independent broker called **Small Action**.

## Features

- **Browse Materials** — search and filter available offerings and material requests
- **Offer Materials** — list surplus or rescued materials for others to discover
- **Request Materials** — post what you're looking for
- **Connect via Small Action** — reach out about any listing through the trusted broker
- **About** — learn about the platform and how it differs from general marketplaces

## Tech stack

- [React 19](https://react.dev/) + [Vite](https://vite.dev/)
- CSS Modules for scoped styling
- React Context for in-memory state (no backend required for MVP)

## Getting started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
